# 字节跳动

> 中国互联网科技公司,Seed团队开发了UI-TARS等Agent控制PC的开源项目

## 基本信息

- **公司性质**: 中国互联网科技公司
- **Agent开发团队**: Seed团队
- **开源项目**: UI-TARS-desktop (33K+ GitHub Stars)

## Agent控制PC的贡献

### UI-TARS项目
<!-- confidence: EXTRACTED -->
- 开发了UI-TARS视觉语言模型(基于Seed-1.5-VL/1.6)
- 发布了UI-TARS Desktop原生GUI Agent应用
- 推出了Agent TARS通用多模态Agent技术栈

### 技术特点
<!-- confidence: EXTRACTED -->
- Apache 2.0开源协议
- TypeScript为主要开发语言
- 支持跨平台(Windows/MacOS/Browser)
- 强调本地私有安全处理
