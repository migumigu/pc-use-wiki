# Refly.AI

> 全球首个 Vibe Workflow 开源平台，通过自然语言驱动的工作流编排降低 AI 工作流搭建门槛

## 核心信息

- **项目类型**：AI 原生工作流平台
- **开源协议**：未知
- **技术栈**：TypeScript（React 前端 + NestJS 后端）
- **GitHub Stars**：约 2.5K-6.3K（报道数据，待官方确认）
- **开源时间**：2025 年末发布，2026 年初正式开源
- **核心创新**：Vibe Workflow 概念，一句话生成工作流

## 核心功能

1. **自然语言工作流**：通过自然语言描述直接生成完整工作流
2. **自由画布设计器**：基于 ReactFlow 的可视化工作流构建界面
3. **多模型调度引擎**：支持 DeepSeek R1、Claude 3.5、Gemini 2.0、OpenAI O3-mini 等 13+ 模型
4. **多线程会话系统**：每条对话保持独立上下文，支持并行任务处理
5. **RAG 检索模块**：支持从 GitHub、Medium、Wikipedia、Arxiv 等平台一键导入内容
6. **MCP 协议集成**：支持与外部工具和服务无缝集成

## 架构特点

| 层级 | 技术实现 |
|------|----------|
| 系统基础层 | NestJS 后端、React 前端、多线程架构 |
| 协议/接口层 | MCP 协议、RESTful API |
| 工具实现层 | 自由画布、多模型调度、RAG 检索 |
| Agent 集成层 | Agent 节点编排、Vibe Workflow |

## 适用场景

- 内容创作自动化（小红书文案、播客、报告、PPT）
- 智能数据分析与报告生成
- 企业知识库问答系统
- AI 创意原型快速验证
- 非技术人员的工作流搭建

## 局限性

- 为降低门槛，主动放弃部分准确性（70% 准确即可接受）
- 主要面向内容生成场景，不适合需要 100% 准确的企业自动化
- Agent 节点成本较高
