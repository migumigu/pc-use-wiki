## 核心价值

codebase-memory-mcp 是一款面向 AI 编程 Agent 的代码智能 MCP 服务器，它将代码库索引成持久化知识图谱，使 Agent 能够高效查询代码结构、调用关系和依赖关系，从而解决大型代码库中的"上下文失忆"问题。

**核心优势**：
- 极致性能（纯 C 实现）
- 原生 MCP 设计
- 零外部依赖部署
- 120 倍 Token 效率提升（vs 直接读文件）

## 技术架构

### 三层架构

```
┌─────────────────────────────────────────┐
│           MCP 协议层                     │
│  工具注册  │  请求处理  │  流式响应       │
├─────────────────────────────────────────┤
│           知识图谱层                     │
│  实体建模  │  关系构建  │  查询引擎       │
├─────────────────────────────────────────┤
│           索引引擎层（纯 C）              │
│  AST 解析  │  符号提取  │  SQLite 存储    │
└─────────────────────────────────────────┘
```

### 关键组件

1. **多语言解析器**
   - 支持 20+ 主流编程语言（声称）
   - 轻量级词法分析 + 语法分析
   - 统一中间表示（IR）

2. **符号提取引擎**
   - 函数/类/变量定义提取
   - 调用关系识别
   - 依赖关系追踪
   - 继承关系建模

3. **知识图谱存储**
   - 嵌入式 SQLite 数据库
   - 实体-关系数据模型
   - 在关系型数据上构建图查询层

4. **MCP 工具集**（约 14 个，据报道）
   - 代码搜索类：语义搜索、模式匹配、相似代码
   - 结构导航类：函数详情、文件结构、项目概览
   - 关系分析类：调用图、依赖关系、引用查找
   - 影响分析类：变更影响、复杂度分析

## 适用场景

1. **MCP 原生工作流用户** — Claude Code、Cursor 等 MCP Agent 用户
2. **大型代码库开发者** — 百万行级代码库的日常开发
3. **性能敏感型用户** — 追求毫秒级响应、低内存占用
4. **AI 编程 Agent 研究者** — 研究 Agent 代码理解能力

## 局限性

1. **语义搜索质量受限** — 相比专用语义平台较弱
2. **动态语言支持不足** — Python/JavaScript 等动态语言分析受限
3. **无运行时信息** — 纯静态分析，无法获取运行时调用链
4. **部分数据待验证** — Star 数、性能数据等需官方确认
5. **生态成熟度较低** — 新项目，社区和工具链尚在建设中
