# inputs

> 跨平台 Python 输入设备库，支持键盘、鼠标和游戏手柄

## 定义

inputs 是一个 Python 库，提供跨平台的键盘、鼠标和游戏手柄访问能力。

## 核心能力

| 能力 | 描述 |
|------|------|
| 设备枚举 | 发现连接的输入设备 |
| 事件监听 | 捕获用户输入事件 |
| 多设备支持 | 同时监听多个同类设备 |
| 游戏手柄支持 | Xbox 360、PS3 等控制器 |

## 技术特点

- **语言**：Python
- **许可证**：BSD-3-Clause
- **平台**：Windows、Linux、macOS、Raspberry Pi
- **安装**：`pip install inputs`

## 平台支持

| 平台 | 支持程度 |
|------|----------|
| Windows | 完全支持 |
| Linux | 完全支持 |
| macOS | 有限支持（High Sierra+） |
| Raspberry Pi | 完全支持 |

## 适用场景

- 游戏手柄控制
- 键盘事件监听
- 多设备输入处理
