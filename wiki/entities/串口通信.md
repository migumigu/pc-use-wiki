# 串口通信

> UART 串行通信协议，是传感器和嵌入式设备与 PC 通信的基础接口

## 技术定义

串口通信（Serial Communication）是一种异步串行通信协议（UART），常用于传感器、嵌入式设备与 PC 的数据传输。在 AI Agent 场景中，串口通信是**传感器数据采集**和**嵌入式设备控制**的基础接口。

## 协议参数

| 参数 | 说明 | 常用值 |
|------|------|--------|
| **波特率** | 数据传输速率 | 9600, 19200, 38400, 115200 |
| **数据位** | 每帧数据位数 | 8 |
| **校验位** | 错误检测 | N（无）、E（偶）、O（奇） |
| **停止位** | 帧结束标志 | 1, 1.5, 2 |
| **流控制** | 数据流调节 | RTS/CTS（硬件）、Xon/Xoff（软件） |

## pySerial 库

pySerial Python 库提供**跨平台串口通信**统一接口，103k+ 项目使用。

### 核心功能

- **跨平台支持**：Windows、Linux、macOS、BSD
- **配置灵活**：波特率、校验位、停止位、流控制
- **文件式 API**：read()、write()、readline()
- **超时控制**：timeout 参数防止阻塞

### 典型代码

```python
import serial

ser = serial.Serial('/dev/ttyUSB0', 9600, timeout=1)
ser.write(b'GET_TEMP\n')
line = ser.readline()  # 读取传感器数据
ser.close()
```

## AI Agent 应用场景

1. **传感器数据采集**：读取温度、湿度、光照等传感器数据
2. **Arduino 控制**：通过串口控制 Arduino MCU 执行任务
3. **嵌入式设备监控**：监控 PLC、MCU 状态并报警
4. **IoT 网关通信**：串口透传 IoT 设备数据到云端

## 设备类型

- **Arduino MCU** — 开源单片机平台
- **传感器模块** — 温度、湿度、光照、加速度等
- **PLC** — 可编程逻辑控制器
- **IoT 网关** — 物联网数据透传设备

## 技术挑战

- **端口枚举**：某些 OS 可能列出不可用端口
- **超时阻塞**：readline() 无超时会永久阻塞
- **数据格式**：不同设备的协议格式不一致
- **驱动安装**：USB-Serial 需要安装驱动
