# AutoHotkey

> Windows 平台的桌面自动化脚本语言，支持热键、热字符串、窗口管理

## 定义

AutoHotkey 是一款免费开源的 Windows 桌面自动化工具，通过自定义脚本语言实现键盘热键、文本自动替换、窗口管理等功能。

## 核心能力

| 能力 | 描述 |
|------|------|
| 热键定义 | 支持 Ctrl、Alt、Win、Shift 组合键 |
| 热字符串 | 自动替换缩写文本 |
| 键盘模拟 | 发送任意键序列 |
| 鼠标控制 | 移动、点击、滚轮操作 |
| 窗口管理 | 激活、关闭、调整窗口大小 |
| 文件操作 | 读写文件、目录管理 |

## 技术特点

- **语言**：C++/C 编写的解释器
- **许可证**：GPL-2.0
- **平台**：仅 Windows
- **最新版本**：v2.0.26（May 4, 2026）

## 版本状态

- **v1**：已停止维护，仅社区支持
- **v2**：当前主版本，持续更新中

## 适用场景

- 办公效率提升（文本替换、快捷键）
- 重复性任务自动化
- 自定义键盘布局
- 窗口管理脚本

## 局限性

- 仅限 Windows 平台
- GUI 识别能力有限，主要基于坐标和窗口标题
- 游戏环境中可能被反作弊系统检测
