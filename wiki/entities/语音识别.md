# 语音识别

> 将语音转换为文本的技术，是 AI Agent 多模态输入的核心能力

## 技术定义

语音识别（Speech Recognition）是将人类语音信号转换为文本的技术。在 AI Agent 场景中，语音识别是**多模态输入**的关键组成部分，允许 Agent 通过语音指令执行任务。

## 技术分类

| 类型 | 代表引擎 | 特点 | 适用场景 |
|------|----------|------|----------|
| **离线引擎** | CMU Sphinx, Vosk, Whisper | 无需网络，隐私保护 | 嵌入式设备、隐私场景 |
| **在线 API** | Google Cloud Speech, Azure Speech | 高精度，需网络 | 企业应用、高精度场景 |
| **现代模型** | OpenAI Whisper, Groq Whisper | 最新技术，速度快 | 实时转写、多语言 |

## SpeechRecognition 库

SpeechRecognition Python 库提供**多引擎统一接口**，支持 14+ 语音识别引擎/API。

### 核心功能

- **麦克风输入**：通过 PyAudio 实时录音
- **音频文件转写**：支持 WAV/FLAC/AIFF 格式
- **后台监听**：支持后台监听语音指令
- **噪声校准**：自动适应环境噪声

### 典型代码

```python
import speech_recognition as sr

r = sr.Recognizer()
with sr.Microphone() as source:
    r.adjust_for_ambient_noise(source)
    audio = r.listen(source)
    text = r.recognize_google(audio)
```

## AI Agent 应用场景

1. **语音指令识别**：Agent 接收"打开浏览器"、"搜索天气"等语音指令
2. **多模态输入**：语音 + 文本 + 视觉联合理解
3. **会议转写**：实时转写会议录音并生成摘要
4. **嵌入式设备控制**：使用 Vosk 离线引擎控制 IoT 设备

## 技术挑战

- **噪声干扰**：环境噪声影响识别精度
- **多语言支持**：不同语言的模型和词典
- **实时性**：在线 API 的延迟问题
- **隐私安全**：语音数据的存储和传输
