# 传感器

> 物理世界数据采集设备，通过串口、USB、GPIO 等接口与 PC 通信

## 技术定义

传感器（Sensor）是用于检测和测量物理量的设备，如温度、湿度、光照、加速度等。在 AI Agent 场景中，传感器是**物理世界感知**的关键接口，允许 Agent 通过数据采集了解环境状态。

## 传感器类型

| 类型 | 代表传感器 | 输出格式 | 适用场景 |
|------|------------|----------|----------|
| **温度传感器** | DS18B20, DHT11 | 数值（℃） | 环境监控 |
| **湿度传感器** | DHT11, DHT22 | 数值（%） | 空气湿度监控 |
| **光照传感器** | BH1750 | 数值（lux） | 自动照明 |
| **加速度传感器** | MPU6050 | 数值（m/s²） | 移动检测 |
| **气体传感器** | MQ-2, MQ-135 | 数值（ppm） | 空气质量监控 |

## 通信接口

| 接口 | 代表库 | 特点 |
|------|--------|------|
| **串口（UART）** | [[pySerial]] | 最常用，Arduino 默认接口 |
| **USB** | pyusb | 高速数据传输 |
| **GPIO** | RPi.GPIO（Raspberry Pi） | 直接连接传感器模块 |
| **I2C** | smbus（Linux） | 多传感器共享总线 |
| **SPI** | spidev（Linux） | 高速传感器通信 |

## AI Agent 应用场景

1. **环境监控**：Agent 监控温度、湿度、光照并自动调节空调、灯光
2. **安防监控**：气体传感器检测火灾、煤气泄漏并报警
3. **健康监控**：心率、血压传感器数据采集并分析
4. **工业监控**：PLC 传感器数据监控设备状态

## 数据采集流程

```python
import serial

# 打开串口（Arduino）
ser = serial.Serial('/dev/ttyUSB0', 9600, timeout=1)

# 发送命令
ser.write(b'GET_TEMP\n')

# 读取传感器数据
line = ser.readline()  # "TEMP:25.5\n"
temp = float(line.split(':')[1])

ser.close()
```
