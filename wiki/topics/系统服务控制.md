# 系统服务控制

> AI Agent控制系统服务和进程的技术栈,涵盖进程管理、网络配置、权限控制、系统监控

## 定义

<!-- confidence: INFERRED -->
系统服务控制是AI Agent通过脚本、API、命令等方式管理系统进程、服务、网络、权限等系统级资源的能力。

## 技术栈分层

### 系统基础层
<!-- confidence: INFERRED -->
- **Windows API**: 系统级接口调用
- **WMI**: Windows管理规范（系统信息查询标准接口）
- **COM**: 组件对象模型
- **内核接口**: Linux system calls
- **systemd**: Linux PID 1 服务管理器

### 协议/接口层
<!-- confidence: EXTRACTED -->
- **PowerShell**: Windows脚本化管理接口
- **SSH**: Unix/Linux远程控制协议
- **WinRM**: Windows远程管理协议

### 工具实现层
<!-- confidence: EXTRACTED -->
- **PowerShell**: 系统管理脚本框架（Windows）
- **systemd**: Linux 服务管理器（主流 init 系统）
- **Bash**: Unix/Linux命令行工具
- **Python os/sys**: 系统调用库
- **pywin32**: Python Windows API 扩展（底层Windows API调用）
- **psutil**: 跨平台进程和系统监控库（Python）
- **Prefect**: Python 工作流编排框架（任务调度与监控）
- **pystemd**: systemd Python 集成库（D-Bus 服务管理）
- **GPUtil**: NVIDIA GPU 状态获取库（Python，解析nvidia-smi）
- **netifaces**: 跨平台网络接口信息获取库（C扩展）
- **py-cpuinfo**: 纯Python CPU信息检测库（多架构支持）

### Agent集成层
<!-- confidence: INFERRED -->
- **Open Interpreter**: 代码执行环境调用PowerShell
- **MCP PowerShell Server**: 模型上下文协议集成
- **Function Calling**: Agent调用系统管理函数

## 核心技术方法

### 进程管理
<!-- confidence: EXTRACTED -->
- **监控**: 获取进程列表、CPU/内存使用
- **控制**: 启动、停止、重启进程
- **调度**: 进程优先级、资源限制

### 服务管理
<!-- confidence: EXTRACTED -->
- **状态查询**: 服务运行状态、启动类型
- **启停控制**: 启动、停止、重启服务
- **配置管理**: 服务启动类型、依赖关系

### 网络控制
<!-- confidence: EXTRACTED -->
- **配置管理**: IP地址、DNS、网关设置
- **连接测试**: 网络连通性检测
- **防火墙规则**: 端口开放、规则管理

### 权限管理
<!-- confidence: INFERRED -->
- **用户管理**: 创建、删除、权限设置
- **文件权限**: ACL、权限继承
- **策略配置**: 执行策略、安全策略

## 代表性工具对比

| 工具 | 平台 | 核心能力 | Agent集成方式 |
|------|------|---------|--------------|
| PowerShell | Win/Linux/Mac | 系统管理脚本 | 代码执行调用 |
| Bash | Unix/Linux | 命令行工具 | SSH远程执行 |
| WMI | Windows | 系统信息查询、服务管理 | PowerShell封装/pywin32 COM |
| Python os/sys | 跨平台 | 系统调用库 | 代码执行 |
| pywin32 | Windows | 底层Windows API、COM互操作 | Python直接调用 |
| psutil | 跨平台 | 进程监控、系统信息 | Python直接调用，Agent集成简单 |
| Prefect | 跨平台（Python） | 工作流编排、任务调度 | Python decorators，Flow.serve部署 |
| pystemd | Linux | systemd 服务管理、D-Bus | Python直接调用，Unit/Manager类 |
| GPUtil | Linux/Windows | NVIDIA GPU状态监控、可用性检测 | Python直接调用，依赖nvidia-smi |
| netifaces | 跨平台 | 网络接口地址、网关信息 | Python直接调用，C扩展 |
| py-cpuinfo | 跨平台 | CPU信息检测、硬件能力识别 | Python直接调用，纯Python |
