# AutoHotkey GitHub README

> Windows 平台的桌面自动化脚本语言，支持热键、热字符串、窗口管理等功能

## 核心要点

AutoHotkey 是一款免费开源的 Windows 桌面自动化工具：

- **热键系统**：定义键盘快捷键触发自动化动作
- **热字符串**：自动替换缩写文本
- **脚本语言**：自定义脚本语言，语法简单
- **窗口管理**：激活、关闭、调整窗口大小
- **鼠标控制**：模拟鼠标移动和点击

## 技术细节

- **语言**：C++/C
- **许可证**：GPL-2.0
- **最新版本**：v2.0.26（May 4, 2026）
- **平台**：仅 Windows
