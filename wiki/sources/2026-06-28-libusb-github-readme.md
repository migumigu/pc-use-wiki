# libusb GitHub README

> 跨平台 USB 设备访问库，支持 Linux、macOS、Windows 等

## 核心要点

- **跨平台**：Linux、macOS、Windows、Android、WebAssembly
- **USB 协议**：USB 1.0 到 USB 4.0
- **传输类型**：控制、批量、中断、同步传输
- **热插拔支持**：设备连接/断开检测

## 技术细节

- **语言**：C/C++
- **许可证**：LGPL-2.1
- **最新版本**：v1.0.30（May 17, 2026）
