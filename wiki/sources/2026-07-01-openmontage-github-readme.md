# OpenMontage GitHub README

> 首个开源 Agentic 视频制作系统，将 AI 编码助手变成完整视频工作室

## 摘要

OpenMontage 是首个开源的 Agentic 视频制作系统，将 AI 编码助手（Claude Code、Cursor、Copilot 等）转变为完整的视频工作室。2026年3月开源，截止2026年7月已获得 24K+ Stars。支持 12 条生产流水线、52 个 Python 工具、500+ Agent Skill。

## 核心功能

1. **多路径视频生成**：图像动画化、真实素材混剪、角色动画
2. **全程自动化**：主题研究、脚本生成、素材生成、音乐匹配、字幕烧入
3. **零 API Key 能力**：Piper TTS、Archive.org、Remotion
4. **多 AI 助手支持**：Claude Code、Cursor、Copilot、Windsurf、Codex

## 技术指标

- Stars：24K+（2026年7月数据，增长迅速）
- 流水线：12 条
- Python 工具：52 个
- Agent Skill：500+
- 许可证：AGPL-3.0
