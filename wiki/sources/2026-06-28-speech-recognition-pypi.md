## 可选依赖（Provides-Extra）

- `audio` — 音频处理
- `pocketsphinx` — CMU Sphinx 离线引擎
- `google-cloud` — Google Cloud Speech API
- `whisper-local` — OpenAI Whisper 本地模型
- `faster-whisper` — Faster Whisper 加速版本
- `openai` — OpenAI Whisper API
- `groq` — Groq Whisper API
- `cohere-api` — Cohere Transcribe API
- `assemblyai` — AssemblyAI
- `vosk` — Vosk 离线引擎
- `audio-split` — 音频分割

## 关键价值

- 为 AI Agent 提供**多引擎语音识别**的统一入口
- 支持**离线引擎**（Vosk、Whisper），适合嵌入式设备和隐私场景
- 支持**在线 API**（Google、Azure、OpenAI），适合高精度场景
