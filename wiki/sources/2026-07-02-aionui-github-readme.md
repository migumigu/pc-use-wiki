# AionUi GitHub README

> iOfficeAI 开源 AI Agent Cowork 协同办公平台，24K+ Stars，Apache-2.0，17+ Agent 统一管理

## 核心能力

- **Built-in Agent**：零配置即用，21 个内置专业助手
- **Multi-Agent 模式**：自动检测 17+ CLI Agent，统一管理
- **Team 模式**：Leader+Teammate 多Agent协作，内置 Team MCP Server
- **Office 生成**：PPT (Morph .pptx) / Word (.docx) / Excel (.xlsx)，OfficeCLI 驱动
- **MCP 统一管理**：一次配置同步所有 Agent
- **远程访问**：WebUI + Telegram/Lark/DingTalk/WeChat
- **Cron 定时**：7×24 无人值守自动化
- **YOLO/Full-Auto 模式**：一键自动审批
- **9+ 格式预览**：PDF/Excel/HTML/Word/Diff 等

## 技术架构

- 桌面框架：Electron (electron-vite)
- 前端：React + EUI
- 后端：aionrs (Rust)
- Agent协调：ACP (Agent Communication Protocol)
- 构建：oxc ecosystem
- 版本：v2.1.21
