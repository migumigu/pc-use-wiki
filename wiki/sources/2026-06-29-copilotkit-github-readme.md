# CopilotKit GitHub README

> 一句话摘要：AI Agent 前端基础设施，让 Agent 能直接在你的应用界面里渲染按钮、表单、图表，实现真正的人机协作。

## 核心观点

1. **项目定位**：Build agent-native applications — on any framework, on any surface.

2. **核心价值**：从 React 库演变为多平台 agentic 框架，同一个 Agent 可以驱动 Web 应用、移动应用和团队 Slack 工作空间。

3. **AG-UI 协议**：CopilotKit 是 AG-UI Protocol 的发起方，该协议已被 Google、LangChain、AWS、Microsoft、Mastra、PydanticAI 等采用。

4. **主要功能**：
   - Chat UI — 可定制聊天界面，支持消息流和工具调用
   - Backend Tool Rendering — Agent 调用后端工具，UI 直接在前端渲染
   - Generative UI — Agent 运行时动态生成和更新 UI 组件
   - Shared State — Agent 和 UI 双向同步状态
   - Human-in-the-Loop — Agent 暂停等待用户输入/确认
   - Self-Learning — 通过用户反馈持续强化学习

5. **平台支持**：React/Next.js (GA)、Angular、Vue、React Native、Beta: Slack/MS Teams/Discord/Google Chat
