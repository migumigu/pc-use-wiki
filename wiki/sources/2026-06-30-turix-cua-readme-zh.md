# TuriX-CUA GitHub 中文自述

> TuriX-CUA 官方 GitHub 仓库中文自述文件，详细技术细节和发展历程

## 摘要

TuriX-CUA 中文官方文档，100% 开源，MIT 协议免费用于个人和科研。支持 macOS/Windows/Linux，采用多模型架构（brain/actor/memory/planner）。

## 关键信息

| 项目 | 值 |
|------|-----|
| **中文定位** | AI 驱动的数字牛马 |
| **核心理念** | Talk to your computer, watch it work |
| **隐私声明** | 不收集用户数据 |

## 技术特性

**四角色设计**：
- Brain（大脑）：理解与规划
- Actor（执行者）：具体操作
- Memory（记忆）：任务记忆和压缩
- Planner（规划者）：步骤分解

**Skills 系统**：
- Markdown 格式剧本
- YAML frontmatter 定义
- Planner 选择，Brain 执行
