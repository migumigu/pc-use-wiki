# agency-agents GitHub README

> 122K+ Stars 的 AI Agent 人格集合库，16 个 Division 架构，51+ 专业化 Agent

## 概要

**agency-agents** 是一个完整的 AI Agency 技能库，采用"虚拟机构"组织架构，将 Agent 按职能划分为 16 个 Division（部门）。每个 Agent 都是领域专精、个性驱动、交付导向、生产就绪的 AI 专家。

## Division 架构

### Engineering Division（34+ Agents）
- Frontend/Backend/Mobile/AI/DevOps/Network/Embedded/SRE 等领域
- 特色 Agent：Multi-Agent Systems Architect（拓扑、上下文、信任、失败恢复）

### Design Division（9 Agents）
- UI/UX/Brand/Visual/Whimsy Injector

### Paid Media Division（7 Agents）
- PPC/Search Query/Tracking/Programmatic

### Sales Division（9 Agents）
- Outbound/Discovery/Deal/Proposal/Pipeline

### 其他 Division
Academic、Finance、GIS、Marketing、Product、Project-Management、Security、Spatial-Computing、Specialized、Support、Testing

## Agent 技能规范

### 文件结构
- frontmatter：name/description/color/emoji/vibe/services
- 10 个正文章节：Identity/Mission/Rules/Deliverables/Workflow/Communication/Learning/Metrics/Capabilities

### 设计原则
1. **Strong Personality** - 独特声音和性格
2. **Clear Deliverables** - 具体代码示例和模板
3. **Success Metrics** - 可量化指标
4. **Proven Workflows** - 经过实战验证的流程
5. **Learning Memory** - 模式识别和改进

## 工具链设计

支持 13 个平台：
Claude Code、Cursor、Copilot、Gemini CLI、OpenCode、OpenClaw、Antigravity、Aider、Windsurf、Kimi、Codex、Osaurus、Hermes

## 技术价值

- **Agent Skills 规范**：定义了标准化的 Agent 人格模板
- **Division 分层架构**：企业级 Agent 组织管理范式
- **跨平台兼容**：一键安装到多个 AI Coding Agent 工具
