# Agent-Reach GitHub README

> AI Agent 全网访问脚手架，支持 15+ 平台零 API 费用访问

## 摘要

Agent-Reach 是一个为 AI Agent 提供互联网能力的一站式脚手架项目。通过整合 twitter-cli、rdt-cli、yt-dlp、Jina Reader 等上游工具，让 Agent 能够零成本访问 Twitter/X、Reddit、YouTube、Bilibili、小红书、GitHub 等 15+ 主流网络平台。

## 核心功能

1. **一键安装**：`agent-reach install` 完成所有配置
2. **SKILL.md 集成**：Agent 自动发现技能
3. **环境诊断**：`agent-reach doctor` 检测渠道状态
4. **可插拔架构**：每个平台对应独立 channel 文件
