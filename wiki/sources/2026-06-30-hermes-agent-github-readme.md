# Hermes Agent GitHub README

> Nous Research 自进化 AI Agent，204K+ Stars，MIT 协议

## 素材摘要

本素材是 Hermes Agent 的官方 GitHub README，提供了项目的全面概述。

## 关键信息

- **Stars**: 204,328
- **License**: MIT
- **开发者**: Nous Research
- **文档**: https://hermes-agent.nousresearch.com/docs/

## 核心要点

1. **自进化学习闭环**：唯一内置学习循环的 Agent，从经验中创建技能并自我改进
2. **多平台支持**：Linux, macOS, Windows (早期Beta), WSL2, Termux, Android
3. **多消息平台**：Telegram, Discord, Slack, WhatsApp, Signal 等 20+ 平台
4. **终端后端**：Local, Docker, SSH, Singularity, Modal, Daytona, Vercal Sandbox
5. **60+ 内置工具**
6. **MCP 集成**
7. **Skills 系统**：兼容 agentskills.io 开放标准
8. **记忆系统**：持久化记忆、用户画像、FTS5 会话搜索
9. **Cron 调度**：自然语言自动化任务
10. **从 OpenClaw 迁移**：支持自动迁移配置、记忆、技能

## 与 OpenClaw 的关系

Hermes 支持从 OpenClaw 自动迁移：
- SOUL.md (persona)
- Memories
- Skills
- Command allowlist
- Messaging settings
- API keys
