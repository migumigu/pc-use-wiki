# inputs Python Library GitHub README

> 跨平台 Python 输入设备库，支持键盘、鼠标和游戏手柄

## 核心要点

- **跨平台支持**：Windows、Linux、macOS、Raspberry Pi
- **设备类型**：键盘、鼠标、游戏手柄
- **事件模型**：基于事件的异步监听
- **安装**：`pip install inputs`

## 技术细节

- **语言**：Python
- **许可证**：BSD-3-Clause
- **平台**：Windows、Linux、macOS
