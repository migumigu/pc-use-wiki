# Superpowers GitHub README

> AI Agent技能框架，将专业开发方法论固化为可复用的Skills

## 核心内容

Superpowers 是一个为 AI 编码智能体设计的技能框架和软件开发方法论，核心是14个Markdown文件，定义了完整的开发工作流规范。

## 关键发现

### 支持平台
- Claude Code、Antigravity、Codex App/CLI、Cursor、Factory Droid
- Gemini CLI、GitHub Copilot CLI、Kimi Code、OpenCode、Pi

### 工作流规范
1. **brainstorming** - 需求分析与设计验证
2. **using-git-worktrees** - 隔离开发工作空间
3. **writing-plans** - 任务分解与计划制定
4. **subagent-driven-development** - 子代理开发与审查
5. **test-driven-development** - TDD循环
6. **requesting-code-review** - 代码审查
7. **finishing-a-development-branch** - 分支管理

### 设计哲学
- 测试驱动开发（TDD）
- 系统性优于临时性
- 复杂度降低
- 证据优于主张
