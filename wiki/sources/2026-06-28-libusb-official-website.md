# libusb Official Website

> libusb 官方网站，提供下载、文档、FAQ 等资源

## 核心要点

### 特性

- **便携性**：单一 API 跨平台
- **用户态**：无需特殊权限
- **协议无关**：支持所有 USB 版本

### Python 绑定

- **PyUSB**：最常用的 Python 绑定
- **python-libusb1**：通过 ctypes 封装
- **pywinusb**：Windows 专用
