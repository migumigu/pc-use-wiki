# A2A Protocol GitHub Official Repository

> Google 官方发布的 Agent2Agent (A2A) 协议仓库

## 素材摘要

**发布方**：Google LLC
**许可证**：Apache License 2.0
**最新版本**：v0.3.0（2025年7月）
**开源日期**：2025年4月9日（Cloud Next '25）

## 核心信息

- **协议定位**：开放协议，实现不同 AI Agent 之间的通信和互操作性
- **核心特性**：标准化通信、Agent 发现、灵活交互、丰富数据交换、企业级安全
- **解决的问题**：打破孤岛、复杂协作、开放标准、保持不透明

## A2A 解决的问题

1. **打破孤岛**：连接不同生态系统的 Agent
2. **复杂协作**：多个专业 Agent 协同处理单一 Agent 无法完成的任务
3. **开放标准**：社区驱动，促进创新和广泛采用
4. **保持不透明**：无需共享内部内存、工具或专有逻辑

## 与 MCP 的互补关系

- **MCP**：Agent 到工具/数据源的连接（"我亲自干活，工具辅助"）
- **A2A**：Agent 到 Agent 的协作（"我指挥你干活"，团队协作分工）
