# Eigent 与 AionUi 对比分析

> 腾讯云开发者文章深度对比两款开源 AI Cowork 桌面工具

## 核心对比

| 特性 | Eigent | AionUi |
|------|--------|--------|
| 核心定位 | 多Agent协作自动化平台 | CLI AI工具统一图形化桌面 |
| 底层架构 | CAMEL-AI 框架 | Electron (Win/Mac/Linux) |
| 特色功能 | Agent编排、人在回路 | 9+格式预览、WebUI远程 |
| Office生成 | Document Agent | 内置 OfficeCLI |
| 协议 | Apache-2.0 | Apache-2.0 |

## 关键洞见

- AionUi 免费替代 $100/mo 的 Claude Cowork
- Eigent 的 Multi-Agent Workforce 基于动态任务拆解
- 两者都深度集成 MCP 协议
- 本地部署支持 vLLM/Ollama/LM Studio
