## 核心观点

1. **多语言支持**：Python 51.3%, C# 45.6%，API 一致
2. **生产就绪**：专为从原型到生产的 Agent 系统设计
3. **多 Agent 提供商**：Azure OpenAI、OpenAI、Anthropic Claude、Amazon Bedrock、Google Gemini、Ollama、Microsoft Foundry
4. **编排模式**：Sequential、Concurrent、Handoff、Group Collaboration
5. **Built-in 特性**：检查点持久化、流式处理、人在循环、时间旅行
