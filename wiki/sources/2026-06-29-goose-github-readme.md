# Goose GitHub README

> Linux Foundation AAIF 孵化的开源本地 AI Agent，49.7K+ Stars

## 一句话摘要

Goose 是由 Agentic AI Foundation (AAIF) 孵化的开源本地 AI Agent，使用 Rust 构建，提供桌面应用、CLI 和 API 三种交互形态，支持 15+ LLM Provider 和 70+ MCP 扩展。

## 核心要点

- **多入口交互**：Desktop App (macOS/Linux/Windows) + CLI + 嵌入式 API
- **多 Provider 支持**：Anthropic、OpenAI、Google、Ollama、Azure、Bedrock 等
- **MCP 生态**：通过 Model Context Protocol 连接 70+ 扩展
- **开源治理**：Apache 2.0，Linux Foundation AAIF 孵化
- **技术栈**：Rust + tokio + tree-sitter 多语言解析
