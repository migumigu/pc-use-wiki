## Design Context

### Users
开发者、研究人员和技术爱好者，在桌面端研究 AI Agent 桌面控制技术。他们在工作期间浏览 Wiki，快速查找工具、概念和深度分析报告，通过双链在页面间导航。

### Brand Personality
Clean · Technical · Modern — 像 Claude Code、Vercel Docs 那样的开发者文档体验。专业但不冰冷，精致但不花哨。

### Aesthetic Direction
- **基调**: 极简主义，克制的紫色点缀
- **参考**: Claude Code UI, Stripe Docs, Vercel Docs
- **反参考**: 花哨的渐变、毛玻璃、发光效果
- **主题**: 浅色（light mode），深色辅助线 + 紫色强调

### Design Principles
1. **内容优先** — 所有设计决策服务于可读性和信息密度
2. **克制的强调色** — 紫色仅用于链接、活跃状态、焦点，不滥用
3. **精准的间距** — 4pt 系统，间距有意图，减少视觉噪音
4. **字体即品牌** — 使用有辨识度的正文字体，避免 Inter/Roboto 泛化审美
5. **响应式但有主次** — 宽屏优先，渐进增强移动端
