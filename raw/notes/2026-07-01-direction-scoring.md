## 选定方向

**TOP 1：OpenAI Agents SDK**

### 选择理由

1. **官方背书**：OpenAI 官方发布，权威性高，生态影响力大
2. **契合度高**：属于 Agent 集成层的核心基础设施，与知识库研究范围高度匹配
3. **热度高**：25k+ GitHub Stars，社区活跃
4. **研究空白**：知识库中尚无相关研究
5. **技术深度**：涉及 Harness 架构、沙箱安全、多 Agent 协作等核心概念
6. **时效性**：2026年4月发布，是最新的技术趋势

### 研究重点

1. Harness 架构设计理念
2. 原生沙箱安全机制
3. 多 Agent 协作模式
4. 与其他框架（LangGraph、AutoGen、MAF）的对比
5. 实际应用场景
