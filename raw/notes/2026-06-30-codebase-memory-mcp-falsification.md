- **处理方式**：在报告中明确标注数据不一致，以 GitHub 实际数据为准

- **问题描述**："120 倍 Token 节省"的对比基准是"直接读完整文件"，这在实际使用中很少见

- **问题描述**：14 个 MCP 工具的具体名称是基于同类产品的推断，非官方列表
