## 选定方向

**TOP 1: OpenCV VideoCapture + Windows 视频捕获技术栈**

### 选择理由

1. **热度最高**：OpenCV 76K+ Stars，是世界最流行的开源计算机视觉库
2. **契合度满分**：直接命中"硬件接口控制-摄像头"分类
3. **权威来源充足**：
   - OpenCV 官方文档 (docs.opencv.org)
   - Microsoft Media Foundation 官方文档
   - PyPI opencv-python 官方页面
4. **技术栈完整**：
   - OpenCV 提供统一的跨平台 API
   - 底层对接 Windows DirectShow/Media Foundation
   - Python binding 通过 opencv-python 包提供
5. **研究价值高**：
   - 视觉输入是 Computer Use 和 GUI Agent 的核心能力
   - 理解 VideoCapture 可为多模态 Agent 开发打基础
   - 与现有知识库的"屏幕截图"实体形成互补（摄像头 vs 屏幕）
