# Goose GitHub README

*your native open source AI agent — desktop app, CLI, and API — for code, workflows, and everything in between*

goose is a general-purpose AI agent that runs on your machine. Not just for code — use it for research, writing, automation, data analysis, or anything you need to get done.

## 核心特性

- **多平台**: macOS, Linux, Windows 桌面应用
- **多接口**: 完整 CLI 和可嵌入 API
- **多 Provider**: 15+ 支持商（Anthropic, OpenAI, Google, Ollama, OpenRouter, Azure, Bedrock 等）
- **MCP 扩展**: 通过 Model Context Protocol 连接 70+ 扩展
- **开源治理**: Apache 2.0 许可证，Linux Foundation AAIF 孵化

## 技术栈

- Rust 构建（性能和可移植性）
- 支持 ACP（Agent Client Protocol）接入现有订阅
- tree-sitter 多语言代码解析
