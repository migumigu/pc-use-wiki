# AutoHotkey

AutoHotkey is a free, open source macro-creation and automation software utility that allows users to automate repetitive tasks. It is driven by a custom scripting language that has special provision for defining keyboard shortcuts, otherwise known as hotkeys.

## Key Features

- **Hotkeys**: Define keyboard shortcuts for triggering actions
- **Hotstrings**: Auto-replace text as you type
- **Scripting Language**: Custom language for automation scripts
- **Window Management**: Control windows, activate, resize, close
- **Mouse Control**: Simulate mouse movements and clicks
- **File I/O**: Read/write files
- **System Commands**: Run programs, manage processes

## Platform Support

- **Windows** (Win32, x64)
- Windows XP and newer supported
- v1 is legacy, v2 is actively maintained
